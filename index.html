Triangle Strip Generator

Description:

This is a Javascript based generative web application. Please use the controls on the right to change the triangle strips.

Parameters:

  • Numpoints: The number of points within each strip
  • Stroke: The weight of the stroke
  • R: The amount of red in each strip
    G: The amount of green in each strip
    B: The amount of blue in each strip
    Alpha: The opacity of each strip
    Color: The color of the stroke
    bColor: The background color
